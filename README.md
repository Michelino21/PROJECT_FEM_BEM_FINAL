The aim of this project is merging together FEM and BEM method for a laplacian problem.
I have already a base working code for both method, the aim is to combine those code providing solid mathematics interpretation.
Furthermore will be devoleped also some test case scenario to test the effectivness of the FEM BEM coupled methods.

To run the project you need matlab, all the parameters are set in the run_FEM_BEM.m file in the main directory. If the option SALVATAGGIO is set to 1, all the plots, the workspace and the console output are saved in Documentations/runs/relative path. The relative path depends on mesh dimension, mesh discretization, BEM margin, time. Please note that the worspace could become very big (hundreds of MB)

This project is developed during the course of Comutational Electromagnetics at Politecnico di Milano
